# Contributing to TenzoShare

Thank you for your interest in contributing! Please read the detailed guide at [docs/development/local-development.md](docs/development/local-development.md).

## Quick Summary

1. Fork the repository and create a feature branch from `main`
2. Follow the code style guide
3. Write tests for new functionality (target 80%+ coverage)
4. Run `./scripts/test-all.sh` before submitting
5. Open a pull request with a clear description

## Reporting Issues

Use GitHub Issues. Include steps to reproduce, expected vs. actual behavior, and environment details.
